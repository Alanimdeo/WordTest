# WordTest: 온라인 단어시험

## [단어시험 페이지](https://wt.alan.imdeo.kr)

## 기능

-   단어장 생성
    -   영단어 하나 당 품사 별로 구분 가능
    -   .wt 파일(구조는 Json)으로 저장
-   단어시험
    -   .wt 파일을 불러와 랜덤으로 순서를 섞은 후 시험

## 빌드 방법

1. repository clone

```sh
git clone https://github.com/alanimdeo/WordTest.git
```

2. 모듈 다운로드

```sh
yarn
```

or

```sh
npm install
```

3. 빌드

```sh
yarn build
```

or

```sh
npm run build
```
